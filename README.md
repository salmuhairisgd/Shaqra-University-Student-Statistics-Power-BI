## Project Overview
An interactive Power BI Dashboard that provides a comprehensive analysis of student demographics at Shaqra University for the academic year 2024. The dashboard tracks student distribution across various colleges, scientific degrees, and gender, offering data-driven insights for academic planning.

## Key Insights
**Demographic Breakdown:** Visualizing the ratio between Male and Female students.

**Academic Achievement:** Analyzing student distribution based on scientific degrees (Bachelor, Higher Diploma, Intermediate Diploma).

**College Analysis:** Identifying the top-performing colleges by student enrollment for each degree type.

**Enrollment Statistics:** A high-level view of total students (5,684) across 13 different colleges.

## Tech Stack
Power BI (Data Visualization & Dashboard Design)

Power Query (Data Cleaning and Transformation)

DAX (Custom Measures for Student Counts and Ratios)
